10. 您如何看待“物联网开发是前端工程师的新蓝海”这个观点？

关于这个观点，我觉得这里有一个误区：写 JavaScript 的程序员就是前端工程师。前端工程师一定是 JavaScript 程序员，但是 JavaScript 程序员不一定是前端工程师。

物联网开发不仅是前端工程师的新蓝海，还是硬件工程师、后台工程师的新蓝海，只是前端开发者有机会去接触更大的领域。
